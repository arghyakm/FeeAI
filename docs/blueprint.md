# **App Name**: FeeAI

## Core Features:

- AI Fee Assistant: Leverage the Gemini API to respond to student fee-related inquiries, providing instant and personalized support.
- Contextual Chat Tool: The LLM is enabled with a 'tool' such that it will remember previous queries and replies within the current session, to offer personalized and context-aware answers.
- Student Fee Dashboard: Display a comprehensive overview of a student's financial standing, including total fees, paid amount, pending dues, and the upcoming due date.
- Payment History Visualization: Present the last 5 payments in an organized table format, complemented by intuitive charts (using Chart.js) to illustrate payment trends.
- Fee Structure Breakdown: Detail the composition of fees in INR, breaking down costs into categories like tuition, library, and lab fees, enhancing transparency.
- Razorpay Payment Gateway: Integrate Razorpay's test mode to facilitate secure fee payments with real-time validation and clear success/failure notifications.
- Firebase Authentication: Implement a secure authentication system with email/password login and registration, safeguarding access to sensitive dashboard and chat functionalities.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), for trust and clarity.
- Background color: Very light blue (#F0F9FF) to give a clean and professional feel.
- Accent color: Purple (#7E22CE), to create visual interest and highlight calls to action.
- Headline font: 'Space Grotesk', a modern sans-serif for headlines. Body font: 'Inter', for clarity.
- Code font: 'Source Code Pro' for displaying code snippets.
- Employ relevant icons that represent various fee categories, dashboard elements, and payment methods. Use a consistent style.
- Design the layout for a great Indian touch with greetings on login page such as 'Namaste! Apna account login karein ya naya account banayein.'
